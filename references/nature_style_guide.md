# Nature期刊投稿风格指南

> Nature Journal Submission Style Guide

---

## 一、期刊定位

Nature是**顶级综合科学期刊**，影响因子高，接受率低（<8%）。

**目标读者**：跨学科科学家

**写作原则**：让非本领域专家也能理解

---

## 二、文章结构

### Letters（研究论文）

| 部分 | 字数 | 内容 |
|------|------|------|
| 标题 | ≤15词 | 描述主要发现 |
| 摘要 | ≤150词 | 非结构化，单段 |
| 正文 | ≤3000词 | 不分章节编号 |
| 图表 | ≤4个 | 核心数据 |
| 参考文献 | ≤30条 | 关键文献 |

### Articles（长篇研究）

| 部分 | 字数 | 内容 |
|------|------|------|
| 标题 | ≤15词 | 描述主要发现 |
| 摘要 | ≤150词 | 非结构化，单段 |
| 正文 | ≤5000词 | 可分章节 |
| 图表 | ≤6个 | 核心数据 |
| 参考文献 | ≤50条 | 关键文献 |

---

## 三、写作风格

### 语言特点

| 特点 | 说明 |
|------|------|
| **主动语态** | We found that... 而非 It was found that... |
| **短句** | 平均15-20词 |
| **简洁** | 无冗余词汇 |
| **结果优先** | 开门见山说发现 |

---

### 标题风格

| ❌ 差 | ✅ 好 |
|-------|-------|
| A study on the effect of X on Y | X enhances Y through novel mechanism |
| Investigation of X using Y method | X reveals unexpected role in Y |

**原则**：
- 说明发现，不是说明做了什么
- 不要用"A study of..."、"Investigation of..."
- 吸引眼球但不夸张

---

### 摘要风格

**结构**：
```
[1-2句背景] [问题的重要性]. 
Here we show that [核心发现]. 
[方法简述]. 
[关键结果1], and [关键结果2]. 
[机制解释]. 
[意义].
```

**示例**：
```
Machine learning has transformed data analysis, but model 
interpretability remains a challenge. Here we show that a 
novel attention mechanism improves both accuracy and 
interpretability. We trained deep neural networks on three 
benchmark datasets. We found that our method achieves 95% 
accuracy, outperforming existing approaches by 10%. The 
attention weights provide interpretable insights into model 
decisions. These results suggest a promising direction for 
developing trustworthy AI systems.
```

---

### 正文风格

**不使用章节编号**：
```
❌ 1. Introduction
❌ 2. Methods

✅ Introduction (作为标题)
✅ Methods (作为标题)
```

**首段要求**：
- 不需要"Introduction"标题
- 首段就是引入部分
- 直接切入主题

---

## 四、图表要求

### 图片要求

| 项目 | 要求 |
|------|------|
| 格式 | TIFF, EPS, PDF |
| 分辨率 | ≥300 dpi |
| 宽度 | 单栏89mm，双栏183mm |
| 字体 | Arial或Helvetica |

### 图例要求

- 图例放在图片下方
- 简洁说明，不超过3行
- 不需要重复正文内容

---

## 五、参考文献格式

### 正文引用

使用数字上标：¹ 或 [1]

### 参考文献列表

```
Author, A. et al. Title of article. Journal Volume, page range (Year).

示例：
Smith, J. et al. Deep learning improves medical diagnosis. 
Nature 15, 234-256 (2020).
```

**特点**：
- 只列第一作者 + et al.
- 标题不用引号
- 期刊名斜体
- 非常简洁

---

## 六、常见拒稿原因（语言相关）

| 原因 | 解决方案 |
|------|----------|
| 摘要不够吸引人 | 开门见山说发现 |
| 方法描述不清 | 提供详细Methods |
| 结果过度解读 | 谨慎下结论 |
| 图表不专业 | 重新设计图表 |
| 参考文献不完整 | 补充关键文献 |

---

## 七、投稿检查清单

- [ ] 标题≤15词，描述主要发现
- [ ] 摘要≤150词，非结构化
- [ ] 正文≤3000/5000词
- [ ] 图表≤4/6个
- [ ] 参考文献≤30/50条
- [ ] 不使用章节编号
- [ ] 主动语态为主
- [ ] 摘要开门见山说发现
