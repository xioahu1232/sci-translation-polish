# 回复审稿人信写作指南

> Response to Reviewers Writing Guide

---

## 核心原则

**态度**：礼貌但坚定，感谢但专业

**目标**：让审稿人相信"我已经认真对待并改进"

---

## 一、回复信结构

```
开头：感谢编辑和审稿人
正文：逐点回复（Point-by-Point）
结尾：总结修改，再次感谢
```

---

## 二、开头模板

### 标准开头

```
Dear Editor and Reviewers,

Thank you for the opportunity to revise our manuscript entitled 
"[Title]" (Manuscript ID: XXX). We appreciate the thoughtful and 
constructive comments, which have helped us improve the quality 
of our paper.

We have carefully addressed all the concerns raised by the reviewers. 
Below, we provide a point-by-point response to each comment. 
All changes in the revised manuscript are highlighted in red.

Sincerely,
[Author names]
```

---

## 三、逐点回复格式

### 格式规范

```
**Reviewer 1**

**Comment 1**: [审稿人原话]

**Response**: [回复内容]

**Changes in manuscript**: [修改内容，页码/行号]

---

**Comment 2**: [审稿人原话]

**Response**: [回复内容]

**Changes in manuscript**: [修改内容]
```

---

## 四、回复类型与模板

### 类型1：完全接受审稿意见

**模板**：
```
**Comment**: [审稿人建议]

**Response**: We agree with the reviewer's suggestion. 
We have revised the manuscript to address this issue.

**Changes**: We have added a new paragraph on page X, lines Y-Z, 
explaining [内容]. The revised text reads:
"[新文本]"
```

---

### 类型2：部分接受，解释原因

**模板**：
```
**Comment**: [审稿人建议]

**Response**: We appreciate the reviewer's suggestion. 
However, after careful consideration, we believe that [原因]. 
We have addressed this concern by [折中方案].

**Changes**: We have revised page X, lines Y-Z, to clarify that [内容].
```

---

### 类型3：无法接受，礼貌反驳

**模板**：
```
**Comment**: [审稿人质疑]

**Response**: We respectfully disagree with this comment for 
the following reasons: [原因1]. [原因2]. [原因3]. 
We have added additional discussion to clarify our position.

**Changes**: We have added a paragraph on page X, lines Y-Z, 
to address this concern.
```

**注意**：
- 用 "respectfully" 表示礼貌
- 用具体理由支持你的观点
- 引用文献支持你的论点

---

### 类型4：需要补充实验

**模板**：
```
**Comment**: [审稿人要求补充实验]

**Response**: We thank the reviewer for this suggestion. 
Following the reviewer's advice, we have conducted additional 
experiments to [目的]. The new results show that [发现], 
which supports our original conclusions.

**Changes**: We have added a new section "Additional Experiments" 
on page X, with Figure Y showing the new results.
```

## 五、常用表达

### 感谢类

| 表达 | 使用场景 |
|------|----------|
| We appreciate the reviewer's insightful comments. | 通用 |
| We thank the reviewer for this constructive suggestion. | 有建设性意见 |
| The reviewer raises an important point. | 重要问题 |

---

### 接受类

| 表达 | 使用场景 |
|------|----------|
| We agree with the reviewer's comment. | 完全接受 |
| We have revised the manuscript accordingly. | 已修改 |
| This is an excellent suggestion. | 高度认可 |

---

### 解释类

| 表达 | 使用场景 |
|------|----------|
| We appreciate the reviewer's concern. | 理解但解释 |
| However, we would like to clarify that... | 礼貌解释 |
| We respectfully disagree because... | 礼貌反驳 |

---

### 补充类

| 表达 | 使用场景 |
|------|----------|
| Following the reviewer's suggestion, we have... | 按建议补充 |
| We have added additional analysis to address... | 补充分析 |
| A new experiment was conducted to... | 补充实验 |

## 六、常见场景回复示例

### 场景1：审稿人说方法不够创新

```
**Comment**: The method lacks novelty as it combines existing techniques.

**Response**: We appreciate the reviewer's concern about novelty. 
While we agree that individual components have been used before, 
our contribution lies in the novel combination and the resulting 
performance improvement. Specifically: (1) The integration of 
A and B is novel because... (2) Our approach achieves X% improvement 
over existing methods. We have expanded the discussion on novelty 
in the Introduction (page X, lines Y-Z).

**Changes**: We have revised the Introduction to better highlight 
our novel contributions.
```

---

### 场景2：审稿人说样本量太小

```
**Comment**: The sample size is too small to draw firm conclusions.

**Response**: We acknowledge this limitation. While our sample 
size (n=XX) is modest, it is comparable to similar studies in 
this field [references]. The observed effect size was large 
(d=X), providing reasonable statistical power. We have added 
a discussion of this limitation and suggested future studies 
with larger samples (page X, lines Y-Z).

**Changes**: We have added a paragraph discussing sample size 
limitations in the Discussion section.
```

---

### 场景3：审稿人要求引用他们的文章

```
**Comment**: The authors should cite [Reviewer's paper].

**Response**: We thank the reviewer for bringing this relevant 
work to our attention. We have added the citation and discussed 
its relevance to our study (page X, lines Y-Z).

**Changes**: We have cited [Reviewer's paper] and discussed 
its findings in the Literature Review section.
```

## 七、回复信检查清单

- [ ] 每条意见都有回复
- [ ] 回复语气礼貌专业
- [ ] 具体说明修改位置（页码、行号）
- [ ] 引用支持论点的文献
- [ ] 没有遗漏任何意见
- [ ] 字数适中（不过长不过短）
- [ ] 格式清晰（审稿人原话 vs 回复区分开）

---

## 八、常见错误

| 错误 | 后果 | 改正 |
|------|------|------|
| 语气强硬 | 激怒审稿人 | 用 "respectfully" |
| 遗漏意见 | 审稿人会再次指出 | 逐条检查 |
| 模糊回复 | 审稿人不满意 | 具体说明修改内容 |
| 过度辩解 | 显得不专业 | 简洁有力解释 |
