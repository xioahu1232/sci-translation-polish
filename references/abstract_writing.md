# 学术摘要写作指南

> Academic Abstract Writing Guide

---

## 摘要的重要性

摘要是论文的"门面"：
- 决定审稿人是否继续阅读
- 决定读者是否引用
- 决定搜索引擎是否能找到

**数据**：90%的读者只读摘要，不看全文。

---

## 一、摘要类型

### 结构化摘要（Structured Abstract）

**适用期刊**：医学、心理学、部分社科期刊

**结构**：
```
Background / Context
Objective / Aim
Methods
Results
Conclusions
```

**字数**：通常250-350词

---

### 非结构化摘要（Unstructured Abstract）

**适用期刊**：Nature、Science、大部分工程期刊

**结构**：单一段落，无小标题

**字数**：通常150-250词

---

## 二、摘要四要素

无论哪种格式，摘要必须包含：

| 要素 | 内容 | 比例 |
|------|------|------|
| **背景** | 为什么做这个研究 | 10-15% |
| **目的** | 研究要解决什么问题 | 5-10% |
| **方法** | 怎么做的（关键步骤） | 15-20% |
| **结果** | 发现了什么（最重要） | 40-50% |
| **结论** | 意味着什么 | 10-15% |

---

## 三、结构化摘要模板

### 医学/心理学模板

```markdown
**Background**: [1-2句背景，说明研究重要性]

**Objective**: To investigate/examine/determine... [研究目的]

**Methods**: We conducted a [study design] involving [participants/sample]. 
[关键方法1]. [关键方法2]. [统计分析方法].

**Results**: [核心发现1]. [核心发现2]. [统计结果].

**Conclusions**: These findings suggest that... [结论]. 
[临床/实践意义].

**Keywords**: keyword1; keyword2; keyword3
```

---

### 计算机科学模板

```markdown
**Background**: [技术背景和问题]

**Objective**: This paper proposes... [研究目的/创新点]

**Methods**: We design/develop... [方法概述]. 
[关键技术1]. [关键技术2].

**Results**: Experiments on [dataset] show that... [性能结果]. 
[对比结果].

**Conclusions**: The proposed method... [结论]. 
[应用前景].
```

---

## 四、非结构化摘要模板

### Nature风格模板

```
[1句背景] [问题的重要性]. Here we show that [核心发现]. 
[方法简述]. We found that [关键结果1], and [关键结果2]. 
[机制解释]. These results [意义].
```

**示例**：
```
Machine learning has transformed data analysis, but model interpretability 
remains a challenge. Here we show that a novel attention mechanism can 
improve both accuracy and interpretability. We trained deep neural networks 
on three benchmark datasets. We found that our method achieves 95% accuracy, 
outperforming existing approaches by 10%. The attention weights provide 
interpretable insights into model decisions. These results suggest a 
promising direction for developing trustworthy AI systems.
```

## 五、摘要写作原则

### 要做的事 ✅

| 原则 | 说明 |
|------|------|
| 首句吸引人 | 说明研究重要性 |
| 结果具体化 | 给具体数字（p值、百分比） |
| 方法简洁 | 只写关键步骤 |
| 结论有力度 | 说明意义，不只是"we found" |
| 关键词恰当 | 便于检索 |

### 不要做的事 ❌

| 问题 | 原因 |
|------|------|
| 引用文献 | 摘要应独立 |
| 使用缩写（未定义） | 读者可能不理解 |
| 过度技术细节 | 摘要是给非专家看的 |
| 夸大结果 | 学术诚信 |
| 复制正文句子 | 摘要应重新组织语言 |

## 六、中英文摘要差异

### 中文摘要常见问题

| 问题 | 中文习惯 | 英文改进 |
|------|----------|----------|
| 背景 | 大量铺垫 | 1-2句足够 |
| 方法 | 详细描述流程 | 只写关键步骤 |
| 结果 | "取得良好效果" | 给具体数字 |
| 结论 | "有一定意义" | 说明具体意义 |

### 示例对比

**❌ 翻译腔**：
```
In this paper, we studied the effect of X on Y. Through experiments, 
we obtained some meaningful conclusions. The results show that the 
method has good performance.
```

**✅ Native English**：
```
This study examined the effect of X on Y. Experiments on 500 samples 
demonstrated that the method achieved 92% accuracy, outperforming 
existing approaches by 15%. These findings suggest that X significantly 
influences Y, with implications for [specific application].
```

## 七、摘要检查清单

完成摘要后检查：

- [ ] 字数符合期刊要求
- [ ] 格式符合期刊要求（结构化/非结构化）
- [ ] 首句说明研究重要性
- [ ] 结果部分有具体数字
- [ ] 结论说明研究意义
- [ ] 无未定义的缩写
- [ ] 无引用文献
- [ ] 关键词3-5个，便于检索

## 八、常见句式

### 背景句式

| 句式 | 示例 |
|------|------|
| X is crucial for... | Machine learning is crucial for modern data analysis. |
| Recent studies have shown... | Recent studies have shown that... |
| However, X remains unclear. | However, the mechanism remains unclear. |

### 目的句式

| 句式 | 示例 |
|------|------|
| This study aims to... | This study aims to investigate... |
| We examined... | We examined the relationship between... |
| Here we show that... | Here we show that the method achieves... |

### 结果句式

| 句式 | 示例 |
|------|------|
| We found that... | We found that the accuracy improved by 20%. |
| Results showed... | Results showed a significant difference (p<0.05). |
| The method achieved... | The method achieved 95% accuracy. |

### 结论句式

| 句式 | 示例 |
|------|------|
| These findings suggest... | These findings suggest a potential mechanism. |
| This study provides... | This study provides evidence for... |
| The results have implications for... | The results have implications for clinical practice. |
