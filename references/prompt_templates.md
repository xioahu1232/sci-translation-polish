# 学术翻译提示词模板

> Prompt Templates for Academic Translation

---

## 使用说明

以下模板用于指导AI完成特定类型的学术翻译任务。复制模板后，替换 `[占位符]` 即可使用。

---

## 模板1：全文翻译

```
你是一位专业的学术论文翻译专家。请将以下中文段落翻译成适合 [期刊类型] 发表的英文。

**目标期刊**：[Nature / IEEE / APA / General Academic]

**翻译要求**：
1. 使用三层解耦方法：
   - 先提取语义核心（去除中文修辞）
   - 再重构论证逻辑（PEEL结构）
   - 最后用原生英语写作
2. 不要直译，要用学术英语的表达习惯
3. 参考该期刊的语言风格：
   [Nature: 主动语态，短句，结果优先]
   [IEEE: 正式，被动语态可接受，方法导向]
   [APA: 客观严谨，模糊限制语使用频繁]

**关键术语对照表**：
[术语1] → [英文翻译1]
[术语2] → [英文翻译2]

**待翻译段落**：
[粘贴中文段落]

**输出格式**：
- 语义提取（bullet points）
- 结构化大纲（PEEL标注）
- 最终英文版本
```

## 模板2：摘要翻译

```
你是一位学术论文摘要翻译专家。请将以下中文摘要翻译成英文。

**期刊类型**：[Nature / IEEE / APA]

**摘要要求**：
- 字数：≤150词（Nature）或 ≤250词（APA）
- 格式：[结构化 / 非结构化]
- 风格：开门见山，直接陈述发现

**翻译要点**：
1. 首句：说明研究重要性
2. 中间：核心方法和结果（含具体数字）
3. 结尾：研究意义

**中文摘要**：
[粘贴中文摘要]

**输出**：
- 英文摘要
- 修改说明（如有）
```

## 模板3：讨论部分翻译

```
你是一位学术论文写作专家。请将以下中文讨论部分翻译成英文。

**翻译要求**：
1. 结构：总结发现 → 解释原因 → 对比前人 → 局限 → 展望
2. 使用适当的模糊限制语（suggests, may, indicates）
3. 与前人研究对比时使用：
   - 一致：consistent with...
   - 不同：unlike previous studies...
   - 扩展：we extend the work of...

**关键发现**：
[列出主要发现]

**待翻译段落**：
[粘贴中文讨论部分]

**输出**：
- 英文讨论部分
- 过渡词使用说明
```

## 使用技巧

1. **先确定期刊类型** — 不同期刊风格差异大
2. **提供术语对照表** — 确保术语一致性
3. **明确字数限制** — 摘要、正文都有字数要求
4. **分步骤翻译** — 大段落分批翻译，质量更高
