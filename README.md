# SCI Translation Polish

**Native Academic Writing for International Publication**

## Overview

Convert Chinese academic papers to publication-ready English with a three-layer decoupling method that eliminates translationese. Supports Nature, IEEE, ACM, and APA journal styles.

## Features

- **Three-Layer Decoupling**: Semantic extraction → logical restructuring → native writing
- **4-Level Terminology Framework**: From standard translations to coined terms with definitions
- **Style Adaptation**: Nature, IEEE, ACM, APA format support
- **Quality Checklist**: Automated verification against Chinglish patterns
- **6-Step Execution Workflow**: Pre-processing → Semantic extraction → Logical restructuring → Native writing → Quality verification → Final polish
- **20 Professional Examples**: Medical, CS, Education, Engineering, Social Science
- **12 Reference Documents**: Common errors, terminology database, transition words, hedging strategies, etc.
- **7 Prompt Templates**: Full paper, abstract, discussion, response letter, cover letter, method, results
